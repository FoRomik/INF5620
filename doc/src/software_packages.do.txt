#TITLE: Recommended software packages for INF5620

_Warning: This list is under development!_

To follow the most important examples in the course notes, you need
these software packages:

 * Some text editor like Emacs or Vim for programming
 * LaTeX
 * Version control systems: Git, Mercurial, Subversion
 * "Python 2.7.x": "http://www.python.org/"
 * "NumPy": "http://www.numpy.org/"
 * "SymPy": "http://sympy.org/en/index.html"
 * "Matplotlib": "http://matplotlib.org/"
 * "SciTools": "http://code.google.com/p/scitools" (use Mercurial version)
 * "Nose test framework": "https://nose.readthedocs.org/en/latest/"
 * "SciPy": "http://scipy.org"
 * "IPython": "http://ipython.org/"
 * "ImageMagick": "https://github.com/hplgit/parampool"
 * "PDFCrop": "http://pdfcrop.sourceforge.net/", "PDFtk": "http://www.pdflabs.com/tools/pdftk-the-pdf-toolkit/", "PDFnUP": "http://pdfnup.softpedia.com/"
 * Fortran (77), C, and C++ compilers
 * "Cython": "http://cython.org/"
 * "FEniCS": "http://fenicsproject.org/"
 * "Odespy": "https://github.com/hplgit/odespy"

All these packages and much more can be installed on Ubuntu
by "this script": "https://raw.github.com/hplgit/vagrantbox/master/doc/src/vagrant/src-vagrant/install_rich.sh". We also have a "Vagrant box": "http://hplgit.github.io/vagrantbox/doc/pub/vagrant_box.html" with INF5620 software pre-installed, see our "instructions": "vagrant_inf5620.html" for how to download and operate the box. There is a handy list of "Windows binaries for many Python packages": "http://www.lfd.uci.edu/~gohlke/pythonlibs/".

Some other potentially useful packages:

 * "Mayavi": "http://code.enthought.com/projects/mayavi/"
 * "Gnuplot": "http://www.gnuplot.info/" and "Gnuplot.py Python interface": "http://gnuplot-py.sourceforge.net/"
 * "Doconce": "https://github.com/hplgit/doconce"
 * "Parampool": "https://github.com/hplgit/parampool"
