INF5620 - Numerical methods for partial differential equations

Recommended software packages for INF5620

Warning: This list is under development!

To follow the most important examples in the course notes, you need these software packages:

  • Some text editor like Emacs or Vim for programming
  • LaTeX
  • Version control systems: Git, Mercurial, Subversion
  • Python 2.7.x
  • NumPy
  • SymPy
  • Matplotlib
  • SciTools (use Mercurial version)
  • Nose test framework
  • SciPy
  • IPython
  • ImageMagick
  • PDFCrop, PDFtk, PDFnUP
  • Fortran (77), C, and C++ compilers
  • Cython
  • FEniCS
  • Odespy
All these packages and much more can be installed on Ubuntu by this script. We also have a Vagrant box with INF5620 software pre-installed, see our instructions for how to download and operate the box. There is a handy list of Windows binaries for many Python packages.

Some other potentially useful packages:

  • Mayavi
  • Gnuplot and Gnuplot.py Python interface
  • Doconce
  • Parampool
