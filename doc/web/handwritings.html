Summary of lectures

Summary of lectures

  • Aug 23: about the course; derivation of the Forward Euler, Backward Euler, Crank-Nicolson, and theta-rule schemes for u'=-au ("decay"); demonstration of their performance and possible numerical artifacts. Handwritings: Part 1, Part 2.
  • Aug 30: analysis and understanding of the schemes for u'=-au, use of sympy, about programming and verification, about experiments, Handwritings. See also course notes which are available in its first form.
  • Sep 6: Operator notation for finite difference schemes, extension to u'(t)=-a(t)u(t)+b(t), software engineering. The lecture used a mix of handwritings and brief, slide-style lecture notes (see also the solarized version).
  • Sep 13: other schemes, the general u'=f(u,t) problem, 2nd-order ODE for vibrations ("vib"), the 1D wave equation PDE. Preliminary compact slide-style documents for the decay and vib themes have been generated. For the 1D wave equation, see handwritings. Exercises 24, 25, 26, and 27 are solved by Yapi and his group.
  • Sep 20: Analysis of the scheme for the 1D wave equation, stability, extension to 2D, boundary conditions, variable coefficient (wave velocity). Handwritings. Nina and Ingeborg solved the Leapfrog exercises.
  • Sep 27: More about the wave equation: Neumann boundary condition (du/dn=0), 2D, about the compulsory project. Qualitative behavior of the scheme for different wave shapes and abrupt changes in the wave velocity. About debugging in the project. Implementation: vectorization, Cython code, Fortran code (very briefly). Handwritings.
  • Oct 17: Intro to finite element methods: the approximation problem. "Brief, slide-style lecture notes": HTML and HTML solarized.
  • Oct 25: What you should know about modern hardware and how to program it. Lecture by Dag Sverre Seljebotn. Material is available from GitHub.
  • Nov 1, 14.15-18.00: What is the reason for noise in wave equations with constant wave velocity and how does it relate to the initial condition?. See also the previous numerical dispersion analysis for more detailed mathematics. More about function approximation and the finite element method: definition of finite elements, approximation, element matrices and vectors, assembly, variational formulations, essential and natural boundary conditions, examples from 1D differential equations, extensions to 2D and 3D. See "brief, slide-style lecture notes": HTML and HTML solarized.
  • Nov 8, 14.15-18.00: More about variational formulations and solving PDEs with finite elements. See "brief, slide-style lecture notes": HTML and HTML solarized of the full lecture notes. The diffusion equation solved by finite differences (Forward Euler, Backward Euler, Crank-Nicolson). Time-dependent problems and the finite element method, examples on the diffusion equation: part I and part II. FEniCS (flexible software for finite element computations).
  • Nov 15: Analysis of finite difference schemes for diffusion equations, see slides and notes (PDF or Sphinx). Nonlinear PDEs (finite difference and finite element methods), Picard iteration, Newton's method, calculation of Jacobians, continuation methods, see the first slides in a slide collection, with 1 slide per page, 4 slides per page, or 8 slides per page. An intro to methods for nonlinear differential equations was written by hand.
  • Nov 21 (note the date!!): Variational forms of systems of PDEs, including elasticity. Adjustments of the finite element method for advection(-diffusion) equations, see the slide collection above.
  • Nov 29: Exercises on hand calculations of nonlinear finite element equations (see the slides for formulations of the exercises). Solutions: Exercise 1 and 2, Exercise 3, 4, and 5. The Navier-Stokes equations for fluid flow, see notes and slide version. Topics for the exam.
The oral exam is planned on December 19, 20 and 21.
