    • INF5620
      • Autumn 2013

Summary of lectures

  • Aug 19: about the course (slides); background test: HTML or PDF; derivation of the Forward Euler, Backward Euler, Crank-Nicolson, and theta-rule schemes for u'=-au (slides); operator notation (slides); implementation, verification, and computing errors (slides); demonstrations and numerical artifacts (slides) See course notes, "decay", for textbook material and study guides (slides for lectures).
  • Aug 21:
    • Problem lab 8.15-9.00: Read about Newton's law of cooling and do Exercises 1-3.
    • Lecture 9.15-10.00: analysis and understanding of the schemes for u'=-au (slides).
  • Aug 26: Quick overview of Python software development techniques (slides)
  • Aug 28:
    • Problem lab 8.15-9.00: Set up a GitHub or Bitbucket account for the work with INF5620, other courses and your thesis. A quick intro to Bitbucket, GitHub and Git: HTML for online viewing, PDF for printing
    • Lecture: Generalizations to variable coefficients, method of manufactured solutions, schemes for 1st-order ODEs (slides)
